# kerntau Homepage

个人主页 / Portfolio — V5.0

## 技术栈

- React 19 + TypeScript 5.8
- Three.js — 3D 中国地图交互
- GSAP + Framer Motion — 动效
- Tailwind CSS 4
- Vite 6

## 页面结构

| 页 | 名称 | 内容 |
|---|------|------|
| 1 | Vision | 头像、实时钟、轨道环动画、个人简介 |
| 2 | Trajectory | 个人宣言 + 3D 地图 / 技术栈展示（标签页切换） |
| 3 | Access | 博客、GitHub、邮箱（磁吸悬停效果） |

## 本地运行

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`

## 构建

```bash
npm run build
```

产物输出至 `dist/`。
